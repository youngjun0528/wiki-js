###
- {{dataname}} 으로 정의된 data를 가져와서 보여줌
- HTML 형식을 보여줄 때에는 v-html 지시자 사용
- 3항 연산자 사용 가능
- 연산자를 사용하여 숫자현 연산 또는 문자열 결합 가능

```html
<template>
  <div class="hello">
    <h1>{{ msg }}</h1>
    <h2>{{ author + 10 }}</h2>
    <p>age : {{ age + 10 }}</p>
    <p v-html="article"></p>
    <p>{{ seen ? "yes" : "no" }}</p>
  </div>
</template>

<script>
export default {
  name: "HelloWorld",
  data() {
    return {
      msg: "반갑습니다!",
      author: "bob",
      age: 30,
      article: "<strong>Hello</strong> world",
      seen: true,
    };
  },
};
</script>
```
